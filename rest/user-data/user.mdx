---
title: "Account State"
openapi: "GET /api/user"
---

Full clearinghouse state for a wallet — positions, margin, leverage, unrealized PnL.

## Example

```bash
curl "https://hyper.polynode.dev/api/user?address=0xc64cc00b46150e9274d09efc945c951710c4ef02"
```
