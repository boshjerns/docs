---
title: "Funding Rates"
openapi: "GET /exchange/funding"
---

Returns the full HyperLiquid `metaAndAssetCtxs` response: perpetual metadata and per-asset funding/price/volume context.

The response is a 2-element array: `[0]` is the universe metadata, `[1]` is an array of per-asset context objects.

## Example

```bash
curl -H "x-api-key: YOUR_KEY" https://hyper.polynode.dev/v2/funding-rates
```
