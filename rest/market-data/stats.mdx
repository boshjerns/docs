---
title: "24h Statistics"
openapi: "GET /exchange/stats"
---

24h volume and price change for every asset.

## Example

```bash
curl https://hyper.polynode.dev/exchange/stats
```

```json
[
  {
    "asset": "BTC",
    "volume_usd": "234567890.12",
    "change_pct": 1.23,
    "mark": "71041.5",
    "prev_day_px": "70180.0",
    "updated_at": 1775702972409
  }
]
```
