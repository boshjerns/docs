---
title: "OHLCV Candles"
openapi: "GET /api/candles"
---

OHLCV candles for a single asset and interval.

`timestamp` is the start of the interval bucket in milliseconds.
`volume` is in base asset units.
`trades` is the count of fills aggregated into the candle.

Supported intervals: `1m`, `5m`, `15m`, `1h`, `4h`, `1d`.

## Example

```bash
curl "https://hyper.polynode.dev/api/candles?asset=BTC&interval=1m&limit=10"
```

```json
[
  {
    "open": 71040.0,
    "high": 71045.0,
    "low": 71038.5,
    "close": 71041.5,
    "volume": 12.34567,
    "timestamp": 1775702940000,
    "trades": 47
  }
]
```
