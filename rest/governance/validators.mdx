Active validator set with stake, name, and signer address.

## Example

```bash
curl https://hyper.polynode.dev/exchange/validators
```

```json
[
  {

    "name": "Hyper Foundation 4",
    "blocks_proposed": 0,
    "last_seen": 1775702972409
  }
]
```
